## What Is This?

This is the **creative home** of Gear Games' engineering community — a space where 270+ developers across the studio come together to **build, share, and grow** beyond their daily game projects.

At Gear Games, we don't just ship titles like *WWE Champions*, *Marvel Strike Force*, and *Angry Birds Match* — we invest in the craft of engineering itself. This organization is where that investment lives: a growing collection of internal tools, shared libraries, automation scripts, and experiments born from real engineering challenges.

> *"We don't just build games. We build the tools that build the games."*

---

## Our Spirit

This is not a place of rigid processes or formal gatekeeping. It's a **developer-first community** built on a few simple values:

```
SHIP FREELY      Have an idea? Build it. A prototype is worth more than a proposal.
SHARE OPENLY     If it helped your team, it can help every team. Open your code.
GROW TOGETHER    Senior or junior — every engineer has something to teach and learn.
MOVE FAST        Small tools, fast iterations. Done is better than perfect.
STAY POSITIVE    Constructive feedback only. We lift each other up.
```

## Contributing

This organization belongs to **every engineer at Gear Games**. If you've built something useful — a script, a library, a workflow — it has a home here.

```sh
# The contribution flow
1. Open an issue or discussion to share your idea
2. Fork the relevant repository (or request a new one)
3. Submit a pull request with a clear description
4. Get at least one review from a teammate
5. Merge and ship
```

No bureaucracy. No lengthy approval chains. Just good engineering shared generously.
