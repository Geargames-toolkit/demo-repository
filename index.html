We don’t just build games. We build the tools that build the games.

Geargames Toolkit is the central hub for internal engineering tools at Gear Games. Since 2006, our Hanoi studio has grown into a 270+ engineer powerhouse delivering titles for some of the world’s most recognized IP.

This organization is where we invest in ourselves — sharing scripts, utilities, shared libraries, and engineering standards across every team to move faster, stay consistent, and ship with confidence.

Every tool here is built by engineers who live with the problem. Every contribution raises the floor for the entire organization.

How We Build

The engineering values behind every tool in this organization.

01

Ship Freely

A working prototype beats a perfect proposal. Get it out, get feedback, iterate.

02

Share Openly

If it solved your problem, it will solve someone else's. Make it reusable, make it available.

03

Grow Together

Every engineer has something to teach and something to learn. Contributions from all levels.

04

Move Fast

Small tools. Fast iterations. Done is better than theoretical. Ship, measure, improve.

05

Build with Pride

Code shipped under this org represents the craft of the entire team. Quality is non-negotiable.
